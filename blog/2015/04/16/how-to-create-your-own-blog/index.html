How to Create Your Own Blog

If you want to create your own blog, you can follow these steps:

Clone the files from the repo to your computer:

git clone git://github.com/imathis/octopress.git octopress

Change to the folder:

cd octopress

Download any necessary dependencies:

bundle install

Install:

rake install

Configure with your account information:

rake setup_github_pages

Enter your username instead of mine: https://github.com/railsdevl/railsdevl.github.io

Generate the files:

rake generate

Deploy the code to production:

rake deploy
